# seulsale.github.io

Personal portfolio website for Sergio Saucedo — a Software Developer at IBM specializing in automation, DevOps, and infrastructure.

## About

This site showcases professional experience, projects, and homelab infrastructure. Key sections include:

- **Projects** — Featured work including the Ansible Z Collection for z/OS automation and homelab infrastructure-as-code
- **Tech Stack** — Ansible, Terraform, Python, Go, Bash, Linux, IBM Z, Docker, Kubernetes, and more
- **Homelab** — A 3-node Proxmox cluster running 20+ self-hosted services (Gitea, Grafana, Jellyfin, Home Assistant, etc.)

## Built With

- [Hugo](https://gohugo.io/) — Static site generator
- [Hugo Blox](https://hugoblox.com/) — Theme framework
- [Tailwind CSS](https://tailwindcss.com/) — Styling
- [GitHub Pages](https://pages.github.com/) — Hosting & deployment
